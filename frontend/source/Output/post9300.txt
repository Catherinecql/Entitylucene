How to clone CD or DVD from the CLI so it can be accessed in VirtulaBox Windows VM <body> How can I clone a CD or DVD to an ISO or DMG, using just the CLI, so that I can attach it to a VirtualBox VM.  I often clone CDs or DVDs using the following commands:  precodediskname=`df -kh /Volumes/* | grep -o Volumes.* | cut -d"/" -f2` diskname=`echo $diskname | sed s/\ //g` # remove spaces device=`df -kh /Volumes/* | grep Volumes | cut -d" " -f1`  sudo umount $device dd if=$device of=~/Desktop/$diskname.iso bs=2048 hdiutil eject $device /code/pre  The result is a .ISO on my desktop that is, as far as I can tell, a bit clone of the CD or DVD.  However, when I boot a VirtualBox Windows XP VM and attach the ISO, Windows says it cannot read it.  <answer9330> You can use hdiutil to create the disk image, then convert to a hybrid format:  precodehdiutil create -srcdevice $device $diskname hdiutil makehybrid -o $diskname.iso -joliet -iso $diskname.dmg /code/pre  NOTE: this method does not require unmounting the device first.  <comment9861> That was it. I forgot Windows needs joliet. Thanks.
