How to disable accentuated char suggestion on long click on mac? <body> When pushing a key for a long time, you expect to print a lot of times the same char.  Instead of that, I now have a popup suggesting all accentuated chars.   How to disable that and come back to a classic keybaord function?  <answer31951> Open a terminal (e.g. emTerminal.app/em) and write  precodedefaults write -g ApplePressAndHoldEnabled -bool false /code/pre  then press Enter.  Any open applications will still use the "Press and Hold", but if you quit them and then open them again they should work as expected (use key repeat instead of "Press and Hold").  Reference: http://m10lmac.blogspot.com/2011/07/os-x-107-lion-getting-rid-of-character.html  <answer31955> As given in the link by Tom Gewecke, you can disable the codeCharacter Picker/code by typing the following in a terminal :   precodedefaults write -g ApplePressAndHoldEnabled -bool false /code/pre  <comment36641> Can you please summarize to help future users even if the link breaks?
