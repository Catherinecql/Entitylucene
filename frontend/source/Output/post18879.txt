How to resize finder icons in 10.7 Lion <body> How do I resize icons in Lion's finder. So for example if I wanted to resize the icons in the application folder. In Snow leopard, there used to be a slider at the bottom of the finder page, but thats gone in Lion.  <answer18880> In the menu bar click View - Show Status Bar  img src="https://i.stack.imgur.com/fYU7y.png" alt="enter image description here"  The status bar will appear at the bottom of the Finder window with the icon size slider on the right hand side.  img src="https://i.stack.imgur.com/FLPmF.png" alt="enter image description here"
