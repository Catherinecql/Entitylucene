And deleting the file and focusing Textmate made it disappear from the project pane.  <answer10986> It does indeed work this way, though the project tree can be slow to update if you're adding files on the fly (for example, using "save as...")  <answer11054> Inside a TextMate project, there's two "folder"-ish concepts: Groups (beige icon) and Folders (blue icon).  Groups are things that exist only inside the project, and let you organize references to files however you see fit, without affecting the external location of those files.  Folders are references to actual folders. This is what you want. To add a folder reference to an existing project, just drag the folder from the Finder onto the project drawer.   <answer115595> TextMate doesn't do a good job of creating .jar projects for you.
