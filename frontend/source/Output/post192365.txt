This is why you create a account for yourself as an administrator.  The accepted way, under OS X, to gain root level access is to use the codesudo/code command from the Terminal application. For example, if you want to see the partitioning of your internal drive the command is  precodegpt -r show /dev/disk0 /code/pre  which if entered will result in the following error message.  precodegpt show: unable to open device '/dev/disk0': Permission denied /code/pre  To use the command, you need to use codesudo/code as shown below.  precodesudo gpt -r show /dev/disk0 /code/pre  If you want to become the root user to avoid entering codesudo/code, you can just enter codesudo sh/code. The codeexit/code command can be used to exit from being the root user.  If you want to execute an application as the root user, you can by using the Terminal application. For example, if you want to launch the Finder as the root user, enter the following command.  precodesudo /System/Library/CoreServices/Finder.app/Contents/MacOS/Finder &amp; /code/pre  To avoid the confusion of having two Finder applications open at the same time, it is usually best to quit your Finder application first. This can be done using the following terminal command.  precodeosascript -e 'tell application "Finder" to quit' /code/pre  One word of caution: preceding a command with codesudo/code is not the same as becoming the root user. For example, the commands    precodesudo echo $USER sudo echo $SUDO_USER /code/pre  result in the same output as the commands shown below.  precodeecho $USER echo $SUDO_USER /code/pre  If you become the root user (the superuser), then the same commands result in a different output. This can be verified by entering the commands shown below.  precodesudo sh echo $USER echo $SUDO_USER sudo echo $USER sudo echo $SUDO_USER exit /code/pre  <answer192422> Using your computer logged in as root all the time is like always carrying around all your keys, your passport, $5,000 in cash, that piece of paper with all your passwords written on it and the only photo you have of Flopsy, the adorable rabbit whose death broke your seven-year-old heart.
